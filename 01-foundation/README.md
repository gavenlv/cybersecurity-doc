# 第一阶段：零基础入门（3-6个月）

> 建立扎实的计算机和网络基础，掌握编程技能，理解安全基本概念

## 🎯 阶段目标

完成本阶段学习后，你将：

- ✅ 理解计算机组成原理和操作系统基础
- ✅ 掌握TCP/IP网络协议和网络通信原理
- ✅ 熟练使用Linux命令行和系统管理
- ✅ 掌握Python编程基础
- ✅ 建立网络安全整体认知框架

## 📚 学习模块

### [1. 计算机基础](./computer-basics/)

**学习时间**：2周

**核心内容**：
- 计算机组成原理（CPU、内存、硬盘）
- 二进制、十六进制表示
- 数据编码（ASCII、Unicode）
- 操作系统基础（进程、线程、内存管理）
- 虚拟化技术

**实践任务**：
- 搭建虚拟化实验环境
- 理解计算机工作原理

**学习资源**：
- 《计算机组成原理》
- CS50课程（哈佛大学）

### [2. 网络基础](./networking-fundamentals/)

**学习时间**：3周

**核心内容**：
- OSI七层模型与TCP/IP四层模型
- 常用协议（HTTP、DNS、TCP、UDP）
- IP地址、子网掩码、路由
- 网络设备（路由器、交换机）
- Wireshark抓包分析

**实践任务**：
- 使用Wireshark抓包分析
- 完成网络配置实验

**学习资源**：
- 《计算机网络：自顶向下方法》
- Coursera计算机网络课程

### [3. Linux基础](./linux-basics/)

**学习时间**：3周

**核心内容**：
- Linux发行版选择（Ubuntu、Kali）
- 命令行操作
- 文件系统与权限管理
- Shell脚本编程
- 服务管理

**实践任务**：
- 安装并熟练使用Linux
- 编写Shell脚本
- 完成系统管理任务

**学习资源**：
- 《鸟哥的Linux私房菜》
- Linux基金会认证课程

### [4. 编程入门](./programming-intro/)

**学习时间**：4周

**核心内容**：
- Python基础语法
- 数据结构（列表、字典、集合）
- 文件操作
- 网络编程
- 自动化脚本开发

**实践任务**：
- 编写至少10个Python脚本
- 开发端口扫描器
- 编写自动化工具

**学习资源**：
- 《Python编程：从入门到实践》
- Coursera Python课程

### [5. 安全概念](./security-concepts/)

**学习时间**：2周

**核心内容**：
- 网络安全定义与范畴
- CIA三要素
- 常见攻击类型
- 常见防御措施
- 道德与法律

**实践任务**：
- 了解网络安全法律法规
- 研究安全案例

**学习资源**：
- 《白帽子讲Web安全》
- OWASP资源

## 🎓 学习建议

### 学习方法

1. **理论与实践结合**
   - 每学一个概念立即验证
   - 动手操作是最好的学习方式

2. **循序渐进**
   - 按模块顺序学习
   - 前面知识是后面的基础

3. **建立笔记系统**
   - 记录重点知识点
   - 整理命令和代码示例
   - 记录遇到的问题和解决方法

4. **定期复习**
   - 每周回顾本周内容
   - 每月总结学习成果

### 时间分配

- **工作日**：每天1-2小时
- **周末**：每天3-4小时
- **每周总计**：10-15小时

### 学习资源使用

1. **书籍**：选择1-2本为主，其他为辅
2. **视频课程**：作为主要学习途径
3. **实验平台**：TryHackMe Pre-Security路径
4. **社区**：遇到问题在社区提问

## 💡 学习提示

### 重点内容

1. **网络协议**：理解TCP/IP是学习网络安全的基础
2. **Linux技能**：渗透测试主要在Linux环境进行
3. **Python编程**：自动化脚本开发必备技能
4. **安全思维**：建立安全意识比掌握技术更重要

### 常见问题

**Q: 零基础可以直接学渗透测试吗？**
A: 不建议。渗透测试需要扎实的计算机、网络、编程基础。跳过基础直接学渗透会走很多弯路。

**Q: 需要精通多门编程语言吗？**
A: 初期精通Python一门即可。后期根据需要扩展到C、JavaScript等。

**Q: Linux命令太多记不住怎么办？**
A: 通过大量实践形成肌肉记忆。不必刻意记忆，用多了自然就记住了。

**Q: 这个阶段需要考证吗？**
A: 不需要。这个阶段主要打基础，考证在初级阶段进行。
