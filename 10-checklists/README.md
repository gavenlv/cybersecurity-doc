# 学习检查清单

> 追踪学习进度，确保每个阶段的技能掌握到位

## 🎯 使用说明

### 如何使用

1. **开始学习前**：打印或复制对应阶段的检查清单
2. **学习过程中**：完成一项勾选一项
3. **阶段结束时**：检查是否有遗漏项目
4. **定期回顾**：每月回顾已完成内容

### 评分标准

- ✅ **已掌握**：能够独立完成，无需参考
- ⭕ **部分掌握**：需要偶尔参考文档
- ❌ **未掌握**：完全不熟悉或需要大量参考

## 💡 使用建议

### 学习进度追踪

建议使用以下方式追踪进度：

1. **Excel表格**：创建进度表，定期更新
2. **Notion/Obsidian**：建立知识库和进度追踪
3. **GitHub**：创建私有仓库记录学习笔记
4. **纸质笔记本**：打印清单，手动勾选

### 定期回顾

- **每周**：检查本周学习任务完成情况
- **每月**：回顾本月学习进度，调整计划
- **每季度**：评估阶段学习成果，决定是否进入下一阶段

### 阶段转换标准

从一个阶段进入下一阶段，应该满足：

1. **完成80%以上的检查清单项目**
2. **能够独立完成阶段实践项目**
3. **通过对应的认证考试（可选）**
4. **具备下一阶段所需的前置技能**
