# 学习路线总览

本目录包含网络安全学习路径的核心规划文档，帮助你明确学习方向和评估自身水平。

## 📂 文档说明

### learning-path.md
完整的网络安全学习路径图，包括：
- 五个学习阶段的详细说明
- 每个阶段的学习目标、核心知识点、实践任务
- 阶段转换的标准和建议
- 可视化的学习路径流程图

### skill-matrix.md
网络安全技能矩阵，用于：
- 评估当前技能水平
- 明确技能发展方向
- 制定个性化学习计划
- 跟踪技能成长进度

## 🎯 如何使用

1. **初次学习者**：先阅读 `learning-path.md`，了解整体学习框架
2. **评估水平**：使用 `skill-matrix.md` 进行自我评估
3. **制定计划**：根据评估结果，确定学习起点和重点
4. **定期复盘**：每完成一个阶段，重新评估技能水平

## 📊 学习阶段概览

| 阶段 | 时长 | 核心能力 | 认证目标 |
|------|------|----------|----------|
| 🌱 零基础入门 | 3-6个月 | 计算机基础、网络基础、Linux、编程入门、安全概念 | - |
| 📗 初级 | 6-12个月 | 渗透测试基础、安全运维、安全开发、漏洞分析 | Security+、CEH |
| 📘 中级 | 1-2年 | 高级渗透、应急响应、安全架构、恶意软件分析 | OSCP、CISM |
| 📕 高级 | 2-4年 | 红蓝对抗、安全审计、漏洞利用开发 | CISSP、CISA |
| 🔮 专家级 | 4年以上 | APT研究、安全研究、安全领导力 | OSEE、GXPN |
